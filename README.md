## Specification about what we used and achieved.

***************
### Data

- Semaval16 Turkish data is used as a data.

***************

### Preprocessing

- Zemberek
- NLTK

***************

### Model Training

- Architecture
    - Multi-label Classification with pretrained model BERT.

- Inference Results
    - Accuracy: 0.69
    - Recall: 0.65
